## Overview
This project is a real estate financial model developed as part of a financial analysis assignment involving Gallagher & Mohan.

The model evaluates the financial feasibility of a real estate asset using projected rental income, operating expenses, cash flows, and valuation metrics whic was used to determine a feasible IRR for investment and investment opportunity.

## Key Features
- Assumptions-driven model
- Rental income projections
- Operating cost analysis
- Net Operating Income (NOI) calculation
- Cash flow forecasting
- Valuation using capitalization method

## Tools Used
- Microsoft Excel
- Finance AI
- Financial modeling best practices

## Disclaimer
This model is created for educational and portfolio purposes only. All data has been modified and does not represent real client information.
